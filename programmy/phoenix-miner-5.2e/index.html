    Что нового в Феникс Майнере?

    Исправления в этом выпуске (начиная с 5.2d) включают:

    • Исправлены проблемы с 4 ГБ видеокартами AMD при майнинге ETH или ETC
    • Исправлены проблемы с использованием видеокарт AMD на 8 ГБ
    • Исправлен сбой при использовании Blockchain Beta драйверов AMD для Windows от 2017 года (все еще не рекомендуется использовать эти драйверы)
    • Добавлена поддержка последних драйверов AMD Windows 10 20.11.2.

    Батник для Ethermine

    PhoenixMiner.exe -pool eth-eu2.nanopool.org:9999 -wal 0x360d6f9efea21c82d341504366fd1c2eeea8fa9d -worker Phoenix -epsw x -mode 1 -Rmode 1 -log 0 -mport 0 -etha 0 -retrydelay 1 -ftime 55 -tt 79 -tstop 89 -tstart 79 -fanmin 30 -coin eth -rvram -1 -eres 0 pause
