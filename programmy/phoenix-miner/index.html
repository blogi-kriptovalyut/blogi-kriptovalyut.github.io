    PhoenixMiner ETC

    Для повышения производительности и стабильности майнинга Ethereum Classic используйте следующие команды батника:

    -dagrestart 2 -gser 2

    Пример:

    PhoenixMiner.exe -pool eu1-etc.ethermine.org:4444 -wal 0x7b9cc9270cF06889a4286221931D3a58e627F888 -worker Phoenix -pass x -log 0 -retrydelay 1 -ftime 55 -tt 79 -tstop 89 -tstart 85 -Rmode 1 -fret 1 -rate 1 -dagrestart 2 -gser 2 -eres 0 -coin etc
    pause

    Что нового в версии 5.5c

    Видеокарты AMD с оперативной памятью более 4 ГБ теперь будут работать без проблем до эпохи DAG 500.

    Добавлен новый параметр -mcdag для сброса разгона памяти на картах Nvidia во время генерации DAG.

    Используйте -daglim 1 для автоматической работы или укажите конкретное значение для максимального размера DAG файла (пример: -daglim 4000 установит максимальный размер DAG 4000 МБ)

    Добавлен параметр -rxboost (также доступный через команду -vmr) для повышения производительности карт AMD на базе GDDR5 (Polaris или более ранних версий)

    Добавлена ​​поддержка таймингов памяти для карт AMD Vega. Используйте параметр командной строки -straps, чтобы активировать (пример: -straps 1 активирует самую низкую скорость тайминга)

    Параметр синхронизации памяти Nvidia (тайминг) улучшен и теперь отключен во время генерации DAG, чтобы избежать любых возможных проблем с нестабильностью.

    Множество других мелких улучшений и исправлений

    Команда reboot

    “-r 1” - закрывает майнер и вызывает файл “reboot.bat” (для Линукс - “reboot.bash” или “reboot.sh”) из каталога майнера (если он там есть) если что-то не то с какой-то из видеокарт. Таким образом, вы можете создать файл “reboot.bat” и предпринять какие-то действия, например, перезагрузить компьютер, поместив там строку: “shutdown /r /t 5 /f”, где:

    • r - перезагрузка;
    • t 5 - таймаут завершения работы 5 секунд;
    • f - принудительно завершить программы без предупреждения.

    Пример файла reboot.bat

    shutdown /r /t 10 /f

    rem Здесь:
rem - r - перезагрузить компьютер;
rem - t 10 - таймаут завершения работы 10 секунд;
rem - f - принудительно завершить программы без предупреждения.

    Майнинг на 4 ГБ и DAG файл

    Командой -rvram - 1 можно уменьшить потребление памяти майнером для того, чтобы работали 4 ГБ карты

    Командой -eres 0 можно уменьшить DAG эпоху для резервирования в памяти

    Командой -dagrestart можно перезапустить майнер при выделении буфера для новой DAG эпохи. Возможные значения: 0 - никогда, 1 - всегда, 2 - автоматически (майнер принимает решение в зависимости от версии драйвера). Это относится к картам AMD на 4 ГБ, у которых могут быть проблемы с новыми DAG эпохами после эпохи 350.

    Командой -lidag 3 Можно замедлить генерацию DAG файла

    Требования

    • ОС: Windows 7/8/8.1/10 (только 64-битные системы);
    • Процессор: Любой
    • Видеокарты NVIDIA и AMD (от 3 ГБ для ETC и от 4 ГБ для ETH)

    Феникс Майнер вылетает?

    1. Проверьте синтаксис бат файла. Не должно быть лишних символов, пробелов и запятых.
    2. Обновите драйвера на видеокарту.
    3. Если разгон видеокарты слишком велик, то драйвер выйдет из строя и зависнет компьютер. Снизьте разгон.
    4. Увеличьте файл подкачки.
    5. Добавьте программу в исключения Defender.

    MSI afterburner не работает с новыми драйверами amd для win10?

    Как тогда разогнать? Программа не работает с драйверами 20.4.2.

    Вы можете использовать параметры аппаратного управления PhoenixMiner (-cclock, -cvddc, -mclock и т. д.). Они должны нормально работать со всеми относительно недавними драйверами (то есть с драйверами прошлого года или двух).

    Вы также можете использовать -hstats 2, чтобы увидеть фактические текущие значения разгона и напряжений, чтобы проверить, правильно ли они применяются.
